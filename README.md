# AI Chatbot

A clean, responsive chatbot interface built with **Streamlit** and powered by **Ollama**.

## Features
* **Local Execution:** Runs entirely on your machine using Ollama.
* **Streaming Responses:** Real-time text generation for a smoother experience.
* **Session History:** Keeps track of your conversation within the session.

## Setup
1. **Install Ollama:** Ensure you have Ollama installed and the `llama3.1` model downloaded (`ollama pull llama3.1`).
